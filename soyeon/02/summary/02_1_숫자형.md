# 02-1 숫자형
- 정수형 : 정수를 뜻하는 자료형
```
a = 123
a = -178
a = 0
```
- 실수형 : 소수점이 포함된 숫자
```
a = 1.2
a = -3.45
```
- 8진수와 16진수
  - 8진수
  ```
  a = 0o177
  ```
  - 16진수
  ```
  a = 0x8ff
  b = 0xABC
  ```
- 복소수
```
a = 1+2j
a.real # 1.0 실수 부분
a.imag # 2.0 허수 부분
a.conjugate() # (1-2j) 켤레 복소수
abs(a) # 2.2360679774997898
```
- 나눗셈 후 소수점 아랫자리를 버리는 / 연산자
```
7/4 # 1.75
7//4 # 1
```
